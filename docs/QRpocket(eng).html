To read, store and show QR codes in the application.

Don't you have a time, you want to show your web pages or other URLs.
QR code is easy way to share URLs between smart phones.
But keep QR codes in file system, etc.
QR pocket is easy way to save your URLs.
And read and show QR code easily.
App can open web page with safari.

QR code list is organized by groups.
And store/load on memory list to/from file.

Saved file is accessable file share from Mac/PC.
And QR code list files are transfered by file share.
